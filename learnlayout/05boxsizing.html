When you set box-sizing: border-box; on an element, the padding and border of that element no longer increase its width. Here is the same example as the previous page, but with box-sizing: border-box; on both elements:

I'm smaller... width 500px margin 20px border 10px
And I'm bigger! Width 500px Margin 20px Padding 50px Border 10Px
Since this is so much better, some authors want all elements on all their pages to always work this way. Such authors put the following CSS on their pages: * { -webkit-box-sizing: border-box; -moz-box-sizing: border-box; box-sizing: border-box; }
